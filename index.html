Reflexão e Aplicação Prática

Propriedades CSS utilizadas:

  • display: flex — Para criar layouts flexíveis em linha ou coluna.
  • justify-content — Para alinhar os itens horizontalmente.
  • align-items — Para alinhar verticalmente os itens.
  • flex-direction — Define a direção dos itens (linha ou coluna).
  • flex-wrap — Permite quebra de linha automática dos itens.
  • align-self — Permite alinhar individualmente um item.
  • flex-grow — Define como um item cresce em relação aos outros.

Aplicação prática com Flexbox:

Um exemplo prático seria um menu de navegação responsivo. Com Flexbox, eu criaria uma nav com `display: flex`, distribuiria os itens com `justify-content: space-between` e, em telas menores, mudaria para `flex-direction: column` usando media queries. Isso garante uma boa experiência de navegação em qualquer dispositivo.
